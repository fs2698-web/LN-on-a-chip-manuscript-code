README: Analysis Pipeline for Human Lymph Node-on-a-Chip
This repository contains the custom R and MATLAB scripts used for the bioinformatic analysis, cellular subset quantification, and statistical modeling presented in the manuscript:
"A Human Lymph node-on-a-chip for Personalized Evaluation of Vaccine Immunogenicity"

## Overview
The codebase is designed to integrate multi-modal data from the Lymph Node-on-a-chip (LN-oC) platform, including:

Single-cell RNA sequencing (scRNA-seq): Profiling the transcriptomic landscape of B cells, T cells, and DCs within the biomimetic environment.

On-chip Effluent Analysis: Quantifying immune cell activation kinetics and subset differentiation (e.g., Plasma cells) using automated thresholding.

System Modeling: Correlating on-chip immune metrics with clinical vaccine responses (e.g., serum antibody titers) to evaluate donor-specific immunogenicity.

## Input files
Demo input files are provided in the `data_demo/` folder. These files are intended to demonstrate the workflow and script structure and are not necessarily the full datasets used in the manuscript.

## How to run the demo
### R
Open R or RStudio, set the working directory to the root folder of this package, and run the relevant script in `scripts_R/`.

Example:
```r
source("scripts_R/05_pseudobulk_edgeR_heatmap.R")
```

### MATLAB
Open MATLAB, set the current folder to the root folder of this package, and run the relevant script in `scripts_MATLAB/`.

Example:
```matlab
run("scripts_MATLAB/02_classify_Bcell_subsets.m")
```

## Expected output
Example output files may be written to the `output_example/` folder, including processed result tables, summary statistics, heatmaps, UMAP plots, pseudotime plots, and B-cell subset percentage tables.

## Script descriptions
### scripts_R
umap.R: Generates comparative UMAP visualizations across tissues and conditions (LN, PBMC, Vaccine +/-).
heatmap.R: Visualizes Z-score normalized expression of immune functional genes across donor groups.
DE plot.R: Generates dot plots for differentially expressed genes.
GSEA.R & GSEA selected pathways.R: Performs pathway enrichment using MSigDB Hallmark sets.
AP1 and Tfh score.R: Calculates module scores for Tfh markers and AP-1 activity.
Cellchat.R: Infers ligand-receptor interactions (e.g., DC-T cell communication).

### scripts_MATLAB
Bcell.m: Quantifies B-cell subsets and plasma cell differentiation from chip effluents.
Linear_Regression_Confidence_Intervals.m: Models the synchronization of T-B cell activation.
Scatter_Plot_with_Confidence_Ellipses.m: Clusters donors into High/Mid/Low groups via 90% confidence ellipses.
